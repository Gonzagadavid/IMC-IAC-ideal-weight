IMC e IAC

O que é IMC

IMC é a sigla para índice de massa corporal. Ela é uma medida internacional usada para fazer o cálculo se uma pessoa está no peso ideal, abaixo ou com sobrepeso. Esse é um método fácil e rápido para a avaliação do nível de gordura de cada pessoa, ou seja, é um preditor internacional de obesidade adotado pela Organização Mundial da Saúde (OMS).

O que é IAC

O IAC significa Índice de Adiposidade Corporal. É um novo método que usa o tamanho dos quadris para medir a gordura do corpo e segue a seguinte lógica: Quanto maior forem os quadris em relação à altura, mais acima do peso se está.
